# Travel Agent - 智能天气穿搭助手

## 项目概述

- **类型**: AI Agent 智能助手（类 ChatGPT 闲聊体验）
- **核心功能**:
  1. 自然语言闲聊交互
  2. 天气查询 + 智能穿搭推荐
  3. 天气趋势图表生成与保存
- **目标用户**: 旅行者和日常生活用户

## 技术方案

- **语言**: Python 3.x
- **Agent 框架**: 轻量级对话引擎（基于规则 + 意图识别）
- **天气 API**: WeatherAPI.com 免费接口（或模拟数据）
- **图表生成**: matplotlib
- **交互方式**: 命令行交互（类似 Chat 聊天气泡）

## 功能列表

### 1. 对话引擎
- 意图识别：闲聊、天气查询、穿搭推荐、退出
- 自然语言理解用户意图
- 保持对话上下文

### 2. 天气获取
- 通过城市名获取实时天气（温度、湿度、风速、天气状况）
- 获取未来 7 天天气预报
- 支持模拟数据模式（无需 API Key）

### 3. 穿搭推荐
- 基于温度区间推荐服装类型（上装、下装、鞋子）
- 基于天气状况（晴/雨/雪/风）推荐配件
- 给出综合穿搭建议

### 4. 天气趋势图表
- 生成 7 天温度趋势折线图
- 保存到用户指定目录（默认 `./weather_charts/`）
- 文件命名：`{城市}_{日期}.png`

### 5. CLI 交互界面
- 类似聊天气泡的 UI 显示
- 输入城市名或直接描述需求
- `quit/exit` 退出程序

## 温度穿搭规则

| 温度范围 | 上装 | 下装 | 鞋子 | 配件 |
|---------|------|------|------|------|
| >28°C   | 短袖 | 短裤/薄裙 | 凉鞋/运动鞋 | 太阳镜、防晒霜 |
| 22-28°C | 短袖/薄长袖 | 长裤/裙子 | 运动鞋 | - |
| 15-22°C | 长袖/薄外套 | 长裤 | 运动鞋 | - |
| 5-15°C  | 毛衣/卫衣 | 厚长裤 | 靴子 | 围巾 |
| <5°C    | 羽绒服/厚外套 | 棉裤 | 棉靴 | 手套、帽子、围巾 |

## 天气状况适配

- **晴天**: 建议戴太阳镜
- **雨天**: 建议带伞、穿防水鞋
- **雪天**: 建议防滑鞋、保暖配件
- **大风**: 建议防风外套、帽子
